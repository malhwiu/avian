# Migrating From v0.6 to `main`

This file contains migration guides for breaking changes that have happened on the `main` branch
since the latest release. These guides are evolving and may not be polished yet.

See [migration-guides/README.md](./README.md) and existing entries for information about Avian's
migration guide process and what to put here.

## `Velocity`, `MaxSpeed`, and `Damping`

The `LinearVelocity`, `AngularVelocity`, `MaxLinearSpeed`, `MaxAngularSpeed`, `LinearDamping`,
and `AngularDamping` components have been replaced with `Velocity`, `MaxSpeed`, and `Damping`
respeoctively. The linear and angular components are now `linear` and `angular` properties.

Constructing velocity:

```rust
// Before
LinearVelocity(Vec3::new(x, y, z))

// After (option 1)
Velocity::from_linear(Vec3::new(x, y, z))

// After (option 2)
Velocity::from_linear_xyz(x, y, z)
```

```rust
// Before
(
    LinearVelocity(Vec3::new(x, y, z)),
    AngularVelocity(Vec3::new(x, y, z)),
    // ...
)

// After (option 1)
(
    Velocity {
        linear: Vec3::new(x, y, z),
        angular: Vec3::new(x, y, z),
    },
    // ...
)

// After (option 2)
(
    Velocity::new(Vec3::new(x, y, z), Vec3::new(x, y, z)),
    // ...
)
```

Constructing damping or max speed (only damping is shown here):

```rust
// Before
LinearDamping(0.6)

// After
Damping::from_linear(0.6)
```

```rust
// Before
(
    LinearDamping(0.6),
    AngularDamping(0.6),
    // ...
)

// After (option 1)
(
    Damping {
        linear: 0.6,
        angular: 0.6,
    }
    // ...
)

// After (option 2)
(
    Damping::new(0.6, 0.6),
    // ...
)
```

Modifying velocity:

```rust
// Before
fn accelerate_linear(mut query: Query<&mut LinearVelocity>, time: Res<Time>) {
    let delta_secs = time.delta_secs();
    for mut linear_velocity in &mut query {
        // Accelerate
        linear_velocity.x += 2.0 * delta_secs;
        // Damping
        linear_velocity.0 *= 1.0 / (1.0 + 0.6 * delta_secs)
    }
}

// After
fn accelerate_linear(mut query: Query<&mut Velocity>, time: Res<Time>) {
    let delta_secs = time.delta_secs();
    for mut velocity in &mut query {
        // Accelerate
        velocity.linear.x += 2.0 * delta_secs;
        // Damping
        velocity.linear *= 1.0 / (1.0 + 0.6 * delta_secs)
    }
}
```

The `Velocity` component also has an `at_point` helper, making it easier to compute point velocities:

```rust
// Before
fn point_velocity(query: Query<(&LinearVelocity, &AngularVelocity)>) {
    for (linear_velocity, angular_velocity) in &query {
        // Compute the velocity at a point 1 unit to the right of the center of mass.
        let point_velocity = linear_velocity.0 + angular_velocity.cross(Vec3::new(1.0, 0.0, 0.0));
        println!("Velocity at point: {:?}", point_velocity);
    }
}

fn point_velocity(query: Query<&Velocity>) {
    for velocity in &query {
        // Compute the velocity at a point 1 unit to the right of the center of mass.
        let point_velocity = velocity.at_point(Vec3::new(1.0, 0.0, 0.0));
        println!("Velocity at point: {:?}", point_velocity);
    }
}
```
